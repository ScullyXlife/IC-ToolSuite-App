IC ToolSuite Feature Overview

A consolidated view of the major DayZ Console Server Tools included in IC ToolSuite for Xbox, PS5, and Nitrado workflows.

Core Editing Tools

Types & Economy

  • types.xml editor for nominal/min/lifetime/restock tuning
  • cfgspawnabletypes.xml attachment/cargo/damage editing
  • cfglimitsdefinition.xml categories/tags/usage/value maintenance

Gameplay & Globals

  • cfggameplay.json editor for stamina, movement, building, and vehicle tuning
  • globals.xml editor for economy lifecycle and spawn-control values
  • cfgweather.xml editor for overcast, rain, fog, wind, and storm behavior

Map & Spawn Workflow Tools

  • Spawn Point visual editor with drag/drop map placement
  • Event Spawns map editing for cfgeventspawns.xml
  • Territories editor for mission zone balancing
  • MapGroupProto and EventGroups balancing helpers

Server Integration & Ops

  • Nitrado server tools support through shared controller/API+FTP fallback
  • File Browser editor for remote config management
  • RPT Error Panel for crash/error diagnostics
  • Admin Live Ops workflows for restricted command tooling
