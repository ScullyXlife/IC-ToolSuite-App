# Tool Features (Full Breakdown)

The README keeps things high-level on purpose.

## End-user highlights (current)

- Desktop stable and Android beta artifacts are published from the same release feed.
- Settings includes Account & Device Linking controls:
  - Create Account
  - Sign In / Refresh Session / Sign Out
  - Load Sessions
  - Create Link Code / Link This Device
  - Load Linked Devices
- Auth endpoint is app-managed for end users (not editable in Settings).
